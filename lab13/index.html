Challenge

For this lab, I wanted to use loops and conditionals to print numbers from 1 to 200, but with a twist: Fizz for 3, Buzz for 5, Boom for 7, and combos for multiples. It was fun to see how the logic stacked up.

Problems

I kept forgetting to reset my string variable inside the loop, so the words would stack up. Also, I had to remember to use jQuery for the output.
